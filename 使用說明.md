# Wealth Mosaic · 財富拼圖 — 使用說明

一個追蹤收支、淨資產、投資與退休規劃的個人財務 app。支援中／英文切換，資料只存在你自己的瀏覽器（localStorage），不會上傳任何伺服器。

---

## 一、啟動 app

```bash
cd finance-app
npm install     # 第一次使用才需要
npm run dev
```

啟動後在瀏覽器開啟畫面上顯示的網址（通常是 `http://localhost:5173`）。右上角可切換 **中文 / English**。

---

## 二、快速上手（建議第一次這樣做）

1. 點右上角 **✦ 引導填寫**（或畫面右下角的圓形 ✦ 按鈕）。
2. 跟著問答一題一題回答：年齡、月收入、房租、生活費、存款、投資、負債、退休年齡與目標等。
3. 每題都可以**打字**或按 **🎤 麥克風用說的**。
4. 不想填的題目按 **略過**；想回上一題按 **上一題**。
5. 填完按 **看儀表板** 就會看到完整的財務總覽。

> 想先看看 app 長怎樣？點 **載入範例** 會帶入一組示範資料，看完按 **清空** 就能全部歸零重新開始。

## 三、五個分頁

| 分頁 | 看什麼 |
|------|--------|
| **總覽** | 淨資產、每月結餘、儲蓄率等關鍵數字一眼看完 |
| **現金流** | 每月收入 vs 支出、**預算 vs 實際**、固定收支項目明細 |
| **淨資產** | 資產與負債清單、淨資產走勢 |
| **投資與目標** | 投資部位配置（圓餅圖）、理財目標進度與**達成預估時間** |
| **退休** | 填年齡／退休年齡／退休後月支出，自動試算退休所需金額與缺口 |

## 四、語音輸入

按任何 🎤 麥克風按鈕後開始說話即可。系統聽得懂中文口語數字：

- 「**六萬五**」→ 65,000
- 「大約**三千**」→ 3,000
- 「**一點五萬**」→ 15,000

### 一句講多筆（固定支出 / 投資 / 負債）

在「其他固定支出」「投資」「負債」這幾題，可以**一句話講多筆**，用頓號或「還有 / 跟 / 和」隔開：

- 「保險三千、手機八百、Netflix 兩百九」
- 「台積電五十萬、0050 二十萬」
- 「車貸二十萬、信用卡五萬」

也可以逐筆輸入後按 **新增此筆**，下方會出現一顆顆項目標籤，按標籤上的 ✕ 可移除。

> 提醒：避免在項目名稱裡用到「和／跟」（例如「水和電費」），會被誤切成兩筆。這種情況改用打字最保險。
>
> 語音功能需要瀏覽器支援，建議用 **Chrome 或 Safari**。不支援時會提示你改用打字。

## 五、退休試算

到 **退休** 分頁，填好這三個必填欄位就會開始計算：

- 目前年齡
- 預計退休年齡
- 退休後每月想花多少（用今天的物價填，系統自動計入通膨）

其他欄位（目前退休資產、每月投入、年化報酬率、通膨率、安全提領率）可微調試算假設。按 **帶入目前數字** 可把現有的淨資產等數據自動填入。

---

## 六、備份與還原

- **匯出**：把目前所有資料下載成一個 JSON 檔（檔名類似 `finance-backup-2026-05-31.json`）。建議定期備份，或要換電腦／換瀏覽器時使用。
- **匯入**：選擇先前匯出的 JSON 檔，把資料完整還原。會覆蓋目前的資料，匯入前記得先匯出備份。

---

## 七、常見問題

**Q：資料會不會不見？**
資料存在瀏覽器的 localStorage。清除瀏覽器資料、用無痕模式、或換瀏覽器都會看不到。重要資料請用 **匯出** 留一份備份。

**Q：清空後退休欄位會怎樣？**
按 **清空** 會把所有資料（含退休）全部歸零成空白，可以重新開始填。

**Q：可以多裝置同步嗎？**
目前只存本機。需要同步的話，用匯出／匯入在裝置間搬移即可。
